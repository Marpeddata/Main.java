Metode-signatur

  • Skriv en metode-signatur (og KUN en metodsignatur) hvor navnet på metoden er getOrganisation.
    Metoden tager en int som argument og returnerer en værdi af typen String.
    • Herunder ser du en skabelon for erklæring af en metode. access modifier, return type, method name (parameter list) { // method body }

    Du kan se bort fra access modifier -delen. Der bliver først vigtig når vi kommer lidt dybere ind i det objektorienterede. Udelad den del, eller sæt den til 'public'.
